# elevator-pitch

A Claude Code skill that generates personal elevator pitches for any context.

## Install

```sh
npx skills add marcoskichel/elevator-pitch
```

## Usage

Trigger in Claude Code:

```
/elevator-pitch
```

Or just ask — "generate my elevator pitch for a networking event" — and Claude will pick it up.

## What it does

Asks only what's missing, then generates three versions:

- **30-second** — Problem → Solution → Ask
- **60-second** — Problem → Solution → Proof → Ask
- **Context-specific** — adapted for networking / job interview / investor meeting / conference

Applies research-backed frameworks (PSPA, Present-Past-Future) with rules for hooks, power verbs, proof points, and closes.
